The aim of this web application is to enable the user to create a personalized to-do list so that they can manage their tasks throughout the day without missing any task. It is a very easy to use to-do web app with minimalistic design. Every person who wants to use the app just needs to create an account. The user can login using his username and password. Authentication ensures that no one else can tamper with their list.
